How to use git on windows 10

  1. Download git on this website https://git-scm.com/ and install it
  2. When installing it dont change anything select only default
  3. After installing it go to bash
  4. Type this on git bash to add your details
    git config --global user.name "Your Name"
    git config --global user.email "yourname@example.com"

    Make sure you replace the "Your Name" and "yourname@example.com" with your real details

    To verify that things are working properly, enter these commands and verify whether the output matches your name and email address.

    git config --get user.name
    git config --get user.email
